# 싱글톤 컨테이너
## 웹 애플리케이션과 싱글톤
- 웹 어플리케이션 클라이언트가 많고 요청이 많음 -> 해당 객체 딱 1개만 생성되고, 공유하도록 설계 -> 싱글톤 패턴

## 싱글톤 패턴
- 한 jvm 에서 해당 객체 인스턴스가 하나만 생성되게끔함
- isSameAs 인스턴스 비교 / isEquals 자바 equals랑 같음
